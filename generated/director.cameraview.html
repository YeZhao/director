getWorldPositionAndRay(imagePixel, imageUtime=None)[source]¶

Given an XY image pixel, computes an equivalent ray in the world coordinate system using the camera to local transform at the given imageUtime. If imageUtime is None, then the utime of the most recent image is used.

Returns the camera xyz position in world, and a ray unit vector.
